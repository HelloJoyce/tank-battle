# 🔥 坦克大战 (Tank Battle)

一个使用 HTML5 Canvas 和 JavaScript 开发的经典坦克大战游戏。

## 🎮 在线试玩

https://你的用户名.github.io/tank-battle/

## 🎯 游戏特色

- 🕹️ 经典复古像素风格
- 🎨 精美的游戏画面和粒子特效
- 🔊 流畅的游戏体验
- 📱 支持键盘操作（WASD/方向键 + 空格）
- 🏆 计分系统和多关卡设计
- 💥 爆炸特效和砖块破坏

## 🚀 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格键` | 发射子弹 |

## 🛠️ 本地运行

1. 克隆仓库
```bash
git clone https://github.com/你的用户名/tank-battle.git
```

2. 进入目录
```bash
cd tank-battle
```

3. 打开 `index.html` 文件即可开始游戏

或者直接使用本地服务器：
```bash
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🎯 游戏规则

1. 控制绿色坦克移动和射击
2. 消灭红色敌方坦克获得分数（每辆+100分）
3. 避免被敌人子弹击中
4. 砖墙可以被子弹摧毁，钢墙无法摧毁
5. 每500分升级，敌人会更强更多

## 📝 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- CSS3
